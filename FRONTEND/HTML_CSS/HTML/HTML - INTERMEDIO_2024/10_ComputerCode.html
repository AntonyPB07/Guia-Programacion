(Teclado) Se utiliza para representar la entrada del usuario, como al presionar una tecla o una combinación de teclas. Se suele mostrar en una fuente monoespaciada y resaltada.

Presione Ctrl + C para copiar.



(Salida) Se utiliza para mostrar la salida de un programa o comando. Se suele mostrar en una fuente monoespaciada, pero sin resaltado.

El comando `ls` muestra una lista de archivos en el directorio actual:

        $ ls


(Código) Se utiliza para mostrar fragmentos de código, como un bloque de código fuente o un ejemplo de código HTML. Se suele mostrar en una fuente monoespaciada con resaltado de sintaxis.

Este es un ejemplo de código Python:

def factorial(n): if n == 0: return 1 else: return n * factorial(n-1)


(Variable) Se utiliza para representar variables en código o texto. Se suele mostrar en cursiva o con un estilo diferente.

La variable `x` se define como 5:

x = 5




 (Formato predefinido)      
        Se utiliza para mostrar texto con formato predefinido, conservando los espacios en blanco, tabulaciones y saltos de línea. Se suele mostrar en una fuente monoespaciada.        
        
Este es un ejemplo de texto preformateado:

        
        Este texto
        tiene
        saltos de línea
        y tabulaciones
